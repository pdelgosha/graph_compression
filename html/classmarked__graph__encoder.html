the list of messages present in the graph, stored in an order consistent with message_dict, i.e. for a message m, if messsage_dict[m] = i, then message_list[i] = m. This is constructed in such a way that message_list[message_dict[x]] = x. This is sorted in reverse order so that all * messages (those messages starting with -1) go to the end of the list.
