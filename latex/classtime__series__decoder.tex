\doxysubsection{Detailed Description}
decodes a time series which is basically an array of arbitrary nonnegative integers 

This class is capable of decompressing arrays of nonnegative integers with size n. Upon construction, n must be given. But later, the object is capable of decompressing any sequence with this size, universally. The input would be the output of {\ttfamily \doxylink{classtime__series__decoder}{time\+\_\+series\+\_\+decoder}} class.
