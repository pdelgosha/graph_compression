\doxysubsection{Detailed Description}
Encodes a simple unmarked graph. 

Encodes a simple graph in the set of graphs with a given degree sequence a. Therefore, to construct an encoder object, we need to specify this degree sequence as a vector of int. For instance (in c++11)


\begin{DoxyCode}{0}
\DoxyCodeLine{\mbox{\hyperlink{classgraph__encoder_a7fedc9ace19e34abb32f1851c8597591}{vector<int>}}\ \mbox{\hyperlink{classgraph__encoder_a56eb5cf480ae5c2fca9f3a45f2ffd4f1}{a}}\ =\ \{3,2,2,3\};}
\DoxyCodeLine{\mbox{\hyperlink{classgraph__encoder}{graph\_encoder}}\ \mbox{\hyperlink{classgraph__encoder_a7fedc9ace19e34abb32f1851c8597591}{E}}(\mbox{\hyperlink{classgraph__encoder_a56eb5cf480ae5c2fca9f3a45f2ffd4f1}{a}});}

\end{DoxyCode}


constructs an encode object E which is capable of encoding graphs having 4 nodes with degrees 3, 2, 2, 3 (in order). Hence, assume that we have defined such a graph by giving forward adjacency list\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\mbox{\hyperlink{classgraph}{graph}}\ G(\{1,2,3\},\{3\},\{3\},\{\});}

\end{DoxyCode}


Note that G has a degree sequences which is equal to a. Then, we can use E to encode G as follows\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{\mbox{\hyperlink{classgraph__encoder_a7fedc9ace19e34abb32f1851c8597591}{pair<mpz\_class,\ vector<int>}}\ >\ f\ =\ \mbox{\hyperlink{classgraph__encoder_a7fedc9ace19e34abb32f1851c8597591}{E}}.encode(G);}

\end{DoxyCode}


In this way, the encode converts G to a pair stored in f, where its first part is an integer, and the second part is the array of integers $\tilde{S}$. Later on, we can use f to decode G.
