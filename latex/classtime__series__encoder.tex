\doxysubsection{Detailed Description}
encodes a time series which is basically an array of arbitrary nonnegative integers 

This class is capable of compressing arrays of nonnegative integers with size n. Upon construction, n must be given. But later, the object is capable of compressing any sequence with this size, universally. The output of the compression is an object of type {\ttfamily pair\texorpdfstring{$<$}{<}vector\texorpdfstring{$<$}{<}int\texorpdfstring{$>$}{>}, mpz\+\_\+class\texorpdfstring{$>$}{>}}.
